# Hotel_Management_System
The Hotel Management System is an Android app designed to manage bookings, guests, staff, and expenses. Integrated with Firebase for real-time updates, it simplifies hotel operations by handling reservations, tracking room availability, managing staff roles, and streamlining financial transactions.
